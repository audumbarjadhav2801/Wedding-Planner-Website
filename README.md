# 💍 Wedding Planner Website

A modern and responsive **Wedding Planner Website** built using **HTML, CSS, and JavaScript**. This project showcases an elegant wedding planning interface with attractive layouts, service sections, galleries, pricing plans, blogs, and an enquiry form.

---

## 📌 Features

- 🏠 Beautiful Landing Page
- 💑 About Us Section
- 🎉 Wedding Services
- 🖼️ Image Gallery
- 💰 Pricing Plans
- 👨‍💼 Team Members
- 📝 Blog Section
- 📞 Enquiry/Contact Form
- 📱 Responsive Design
- ✨ Clean and Modern User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

```text
Wedding-Project/

```
